## ✨ Features

### 🔍 Smart Search

* Search movies instantly
* Search by movie title
* Search by director
* Search by actor

### 🎭 Genre Filtering

* Filter movies by genre
* Dynamically generated genres

### ⭐ Rating Filters

* 7+
* 8+
* 8.5+
* 9+

### 🎬 Movie Details Modal

View:

* Plot
* Director
* Actors
* Awards
* IMDb Rating
* Release Year

### ❤️ Favorites

* Save favorite movies
* Stored using Local Storage

### 📺 Watchlist

* Add movies to watch later
* Persistent across browser refreshes

### 📱 Responsive Design

* Mobile Friendly
* Tablet Friendly
* Desktop Friendly
